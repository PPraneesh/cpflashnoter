# Application Style Guide

## Color Palette

### Background Colors
- Primary Background: bg-gradient-to-br from-gray-900 to-gray-800
- Secondary Background: bg-gradient-to-br from-gray-800 to-gray-900
- Tertiary Background: bg-gray-900/50
- Hover Background: bg-blue-600/20 hover:bg-blue-600/30

### Text Colors
- Primary Text: text-gray-200 (Headings and important text)
- Secondary Text: text-gray-300 (Body text)
- Tertiary Text: text-gray-400 (Less important text)
- Accent Text Blue: text-blue-400 (Interactive elements)
- Placeholder Text: placeholder-gray-500

### Border Colors
- Primary Border: border-gray-700/50 (Container borders)
- Accent Border Blue: border-blue-500/50 (Interactive elements)

### Shadow Colors
- Blue Shadow: shadow-blue-500/20

## Typography

### Font Sizes
- Extra Large: text-2xl (Major headings)
- Large: text-xl (Section headings)
- Medium: text-lg (Sub-headings)
- Regular: text-md (Normal text)
- Small: text-sm (Secondary text)
- Extra Small: text-xs (Labels and badges)

### Font Weights
- Bold: font-bold
- Semi Bold: font-semibold
- Medium: font-medium
- Normal: font-normal

## Spacing

### Padding
- Container Padding: p-6
- Button Padding: px-4 py-2
- Small Element Padding: px-3 py-1
- Extra Padding: p-2, p-3, p-4

### Margins
- Between Sections: space-y-6
- Between Elements: gap-4, gap-6
- Small Gaps: gap-2
- Margin Bottom: mb-2, mb-3, mb-4

## Borders

### Border Radius
- Large: rounded-lg (Containers)
- Full: rounded-full (Toggle switches)

### Border Widths
- Default: border (1px)
- Focus: ring-2 (Focus states)

## Effects

### Transitions
- Duration: duration-200
- Timing: ease-in-out
- Properties: transition-all, transition-colors, transition-transform

### Shadows
- Large: shadow-lg
- Extra Large: shadow-xl

### Opacity
- Semi-transparent: opacity-50
- Disabled state: disabled:opacity-50

## Layout

### Grid
- Mobile: grid-cols-1
- Desktop: md:grid-cols-2
- Gap: gap-6

### Flexbox
- Container: flex
- Spacing: justify-between, justify-center, items-center
- Direction: flex-col
- Grow: flex-grow

## Interactive Elements

### Buttons
Primary Button:
```css
px-4 py-2 bg-blue-600/20 text-blue-400 hover:bg-blue-600/30 
border border-blue-500/50 shadow-lg shadow-blue-500/20 
rounded-lg transition-all duration-200 disabled:opacity-50
```

### Toggle Switch
Container:
```css
w-12 h-6 rounded-full p-1 transition-colors duration-200 ease-in-out
bg-blue-500 shadow-lg shadow-blue-500/20
```

Thumb:
```css
w-4 h-4 rounded-full bg-white transform transition-transform duration-200 ease-in-out
translate-x-6 (active) or translate-x-0 (inactive)
```

## Container Styles

### Card Container
```css
bg-gradient-to-br from-gray-800 to-gray-900 rounded-lg 
shadow-lg p-6 border border-gray-700/50
```

### Input Container
```css
bg-gray-900/50 text-gray-200 w-full p-3 
border border-gray-700/50 rounded-lg 
focus:outline-none focus:ring-2 focus:ring-blue-500/50 
focus:border-blue-500/50 transition-all duration-200
```

## Responsive Design

### Breakpoints
- Mobile First: Default styles
- Tablet/Desktop: md:* (768px and above)

### Mobile Navigation
- Fixed bottom navigation
- Simplified layout
- Larger touch targets

## Animation

### Transitions
- All transitions use duration-200
- Smooth easing with ease-in-out
- Transform transitions for movement
- Color transitions for hover states

### Hover Effects
- Scale: hover:scale-105
- Background color changes
- Shadow intensity changes
- Border color changes

## Best Practices

1. Maintain consistent spacing using the defined spacing scale
2. Use semantic color names based on purpose
3. Ensure sufficient color contrast for accessibility
4. Apply consistent border radius across similar elements
5. Use transitions for all interactive elements
6. Maintain consistent padding/margin patterns
7. Follow mobile-first responsive design
8. Use shadows to create depth hierarchy
